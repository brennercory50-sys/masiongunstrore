# Mason Avenue Firearms & Pawn - Project Instructions

## Overview
Multi-department marketplace website for Mason Avenue Firearms & Pawn (347 Mason Ave, Daytona Beach, FL 32117, Phone: (386) 226-4653). Premium dark luxury theme with 5 departments: Firearms, Jewelry, Electronics, Tools, and Pawn Loans.

## Design Language
- **Colors:** True black (#000), cards at #060606, borders at white/[0.06], deep red (#DC2626) accents sparingly, amber accents for pawn/jewelry
- **Typography:** Tight tracking, uppercase labels, gray hierarchy (white → gray-400 → gray-500 → gray-600 → gray-700)
- **Components:** rounded-xl/2xl cards, card-hover effects, Framer Motion animations, hide-scrollbar strips
- **Logo:** Bulldog mascot at `/public/logo.jpg`, used in navbar + footer (intentional dual-use branding)

## Technical Architecture
- **Auth:** NextAuth with credentials provider, JWT strategy. Admin: john@doe.com / johndoe123
- **DB:** PostgreSQL via Prisma. Models: User, InventoryItem (with department field), SellRequest, ItemRequest, PawnLoanRequest
- **File Upload:** S3 presigned URLs via aws-config.ts + s3.ts
- **Notifications:** NOTIF_ID_SELL_FIREARM_SUBMISSION, NOTIF_ID_ITEM_REQUEST_SUBMISSION, NOTIF_ID_PAWN_LOAN_REQUEST

## 3-Layer Inventory System
- In Stock (green) → "Reserve" CTA
- Available to Order (amber) → "Order" CTA
- Request/Source (red) → "Request" CTA

## Department Config
- `/lib/departments.ts` — categories, brands per department
- 4 departments for inventory: firearms, jewelry, electronics, tools
- Pawn Loans is a separate service page at `/pawn`

## Key Pages
- `/` — Hero + department cards + featured/recent items + sell/pawn CTAs + reviews
- `/inventory` — Department tabs, dynamic filters (category/brand change per dept), search
- `/inventory/[id]` — Product detail with department-aware CTAs (FFL compliance only for firearms)
- `/sell` — Multi-department sell form (firearms, jewelry, electronics, tools)
- `/pawn` — Pawn loan request form with photo upload
- `/order` — Special order request form
- `/admin` — Inventory CRUD + sell/item/pawn request tabs with department field

## Seed Data
- 12 firearms + 4 jewelry + 4 electronics + 4 tools = 24 total items
- New department images at `/public/products/` (jewelry-1-4.jpg, electronics-1-4.jpg, tools-1-4.jpg)
